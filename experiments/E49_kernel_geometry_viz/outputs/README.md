# E49 Outputs Directory

This directory contains all outputs from the kernel geometry visualization experiment.

## Structure

```
outputs/
├── README.md                          # This file
├── results/                           # Primary results
│   ├── e49_results.json              # Main results with validation
│   ├── e49_summary.txt               # Human-readable summary
│   └── validation_report.json        # Theory validation outcomes
├── figures/                           # All plots
│   ├── figure_1_kernel_3d.png        # 3D projection of kernel basis
│   ├── figure_2_singular_values.png  # SVD singular value spectrum
│   ├── figure_3_quotient_heatmap.png # Quotient metric visualization
│   └── plots_manifest.json           # Figure descriptions and metadata
├── data/                              # Intermediate data
│   ├── kernel_basis.csv              # 4×2 complex matrix (real,imag columns)
│   ├── singular_values.csv           # 4 singular values from SVD
│   ├── kernel_angles.csv             # Inner products between basis vectors
│   └── quotient_metric.csv           # 2×2 quotient metric Q = ππ†
├── logs/                              # Execution logs
│   └── run_[timestamp].log           # Detailed computation log
└── PROVENANCE.md                      # Where every number came from
```

## Data Formats

### CSV Files
- **Complex numbers**: Stored as separate real and imaginary columns
- **Headers**: First row contains column names
- **Precision**: Full double precision (no truncation)

### JSON Files
- **Complex numbers**: Stored as {"real": float, "imag": float}
- **Pretty printing**: 2-space indentation
- **UTF-8 encoding**

## Usage

Results are generated by running:
```bash
python src/main.py
```

To regenerate specific components:
```bash
python src/compute.py    # Recompute kernel
python src/visualize.py  # Regenerate plots
python src/validate.py   # Rerun validation
```
